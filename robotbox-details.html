robotBox

Project information

  • Category: Software Design
  • Company: Personal project
  • Project date: 30 March, 2018
  • Project URL: github.com/ibonn/robotBox

This project started out of necessity. In 2018 I was taking a course in college where we were learning mobile robotics fundamentals employing ePuck robots. There were many more people enrolled in the course than there were robots available, and many of them were failing, so getting access to a working robot became a difficult task. That is why I started to develop a 2D simulator that would allow us to work with such robot without having one physically and at the same time it would allow us to continue working from home. The simulator is programmed in Visual Basic .NET (it's the language I knew best at that time) and has some limitations and small bugs, but it is more than OK for the time invested in it. In parallel I modified the pyEpuck script, a Python library used to communicate with the robots via bluetooth to work with this simulator via UDP, so minimal changes would have to be done to our code.
