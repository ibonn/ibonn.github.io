Emotional Films data capture application

Project information

  • Category: Software Engineering
  • Company: Emotional Films
  • Project date: 01 January, 2021
  • Project URL: emotional-films.unavarra.es

This application was designed to capture videos of people reacting to different content using 8 Logitech C930e webcams and a Intel® RealSense™ Depth Camera D455. These images would then be used to generate a dataset with which to train our models. Some of the features of this application are:

  • Operator authentication
  • Simoultaneously recording at 1080p/30fps with 8 Logitech webcams + 720p/30fps with Intel® RealSense™ Depth Camera
  • Pulse rate/oxygen saturation recording by means of a bluetooth connected pulse oximeter
  • It is cinnected to 4 Yeelight smart bulbs which randomly change the lighting conditions in order to obtain a higher variability in the data
  • It also randomly modifies and determines the distance at which the user should be placed using the depth sensor of the Intel® RealSense™ Depth Camera
  • Possibility of repeating/invalidating shots
  • The application communicates via websockets with a tablet to display a questionnaire before and after the recording, as well as to show the user an interface in which he/she signs the consent for the treatment of his/her data
  • Secure communication with our API to send the collected data
